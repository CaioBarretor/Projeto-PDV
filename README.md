# 🛒 Sistema de PDV em Python
Este projeto consiste em um sistema de Ponto de Venda (PDV) desenvolvido em Python com foco no aprendizado dos principais conceitos da linguagem. A aplicação funciona inteiramente pelo terminal e simula o fluxo básico de um caixa, permitindo o gerenciamento de produtos e a realização de vendas.

# Funcionalidades:

Cadastro de novos produtos.
Remoção de produtos cadastrados.
Listagem de todos os produtos disponíveis.
Registro de vendas com quantidade de itens.
Exibição dos produtos adicionados à venda.
Cálculo automático do valor total da compra.
Simulação de formas de pagamento.
Tratamento de entradas inválidas utilizando try/except.
Interface organizada para execução no terminal.
Tecnologias utilizadas
Python 3
Biblioteca padrão time
Conceitos aplicados

O principal objetivo deste projeto foi colocar em prática os conhecimentos adquiridos durante os estudos de Python, desenvolvendo uma aplicação funcional e organizada que simulasse um cenário real de utilização.
